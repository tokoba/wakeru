# 行動規範（Contributor Covenant Code of Conduct）

## 私たちの約束

相互尊重を受け入れるオープンで包括的な環境を促進するため、私たちは貢献者とメンテナ全員として、参加者すべてにハラスメントのない体験を提供することを約束します。年齢、体のサイズ、障がいの有無、民族的特性、性別の同一性と表現、経験のレベル、教育、社会経済的地位、国籍、個人的な外見、人種、宗教、性自認にかかわらず、すべての人を歓迎します。

## 私たちの基準

ポジティブな環境を作るために、私たちは次の行動を期待しています：

- 共感的で他者の意見を尊重した会話を使う
- さまざまな視点や経験を肯定的に受け入れる
- 建設的なフィードバックを優雅に受け入れ、それを提供する
- コミュニティにとって最善となることを何であるか配慮する
- 他のコミュニティメンバーに対して共感を示す

許容されない行動の例：

- 性的な言葉や画像の使用、および不当な性的注意または注目
- いじめ、侮辱、軽視されるコメント、および個人的な攻撃や政治的攻撃
- 公的またはプライベートな嫌がらせ
- 明示的な許可なく、他人の住所や電子メールなどの個人情報の公開
- 職場環境と見なされる場合の不当な性的コンテンツやその他の適切でないコンテンツ

## 私たちの責任

プロジェクトメンテナは、行動規範の解釈と実施に対して明確にされ、適正と見なされる基準に従う責任を負います。また、不適切、脅威、攻撃的、有害なコメントのすべてを削除、編集、拒絶する権限を有しています。

プロジェクトメンテナがこの規範を施行しない、あるいは適用しないと判断した場合、プロジェクトリーダーの裁定が最終的なものとなります。

## 適用範囲

この行動規範は、すべてのプロジェクトスペースと、パブリックスペースでの個人としてのプロジェクト代表時に適用されます。また、プロジェクトスペース内外で、コミュニティを代表している際にも適用されます。プロジェクト代表の例は、公式の電子メールアドレスの使用、ソーシャルメディアでの公式アカウントの投稿、オンラインまたはオフラインでのイベントでの指定代理としての行動などです。プロジェクトメンテナは、代表者に対するこの規範の違反を明確に定義および修正する権限を有します。

## 執行

不寛容、軽視、または不適切と思われるコメントの発生を確認した場合、または懸念がある場合は、tokoba（GitHub経由で連絡可能）にご連絡ください。すべての苦情はレビューおよび調査され、必要かつ適切と思われる結果につながります。プロジェクトチームは、イベントの報告者に機密を保持する権利を有します。より具体的な執行方針の詳細は、別途公開される可能性があります。

不従順な行動をとるプロジェクトメンテナまたはメンテナは、プロジェクトリーダーによって判断された他の適切な是正措置を伴い、プロジェクトメンテナシップの地位から一時的または永久的に削除される可能性があります。
